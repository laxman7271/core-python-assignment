Core Python Assignment

This repository contains 7 basic Python programs based on real-world scenarios.
Each program is written using simple logic, functions, and clean structure.

All programs are tested with sample inputs and handle basic edge cases.

Programs Included

1️⃣ E-Commerce Cart System
Calculates total price of items in a cart.
If cart has more than 5 items, a 10% discount is applied.
Handles empty cart case.

2️⃣ Restaurant Menu Management

Allows adding, removing, and checking menu items.
Uses functions for each operation.
Shows updated menu after changes.

3️⃣ Classroom Performance Tracker

Calculates average marks for each student.
Finds the top performer based on highest average.

4️⃣ Movie Ticket Booking System

Manages booked and available seats.
Supports seat booking and cancellation.
Prevents duplicate booking.

5️⃣ Hospital Patient Management

Stores patient records using list of dictionaries.
Searches patients by disease name.

6️⃣ Customer Feedback Analysis

Calculates percentage of positive ratings (4 or 5).
Handles case when no ratings are given.

7️⃣ Taxi Fare Calculation

Calculates fare for multiple trips.
Uses base fare + per kilometer rate.
Prints each trip fare and total fare.
